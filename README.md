# 画像マージツール｜A3横タイプ（2画像自動合成）

ブラウザ上で動作する、**A3横サイズ専用の画像2枚マージツール**です。  
画像を選んでボタンを押すだけで、POP作成に使えるA3横画像が瞬時に完成します。

---

## 🚀 主な機能

- 画像枠（2つ）にファイルを選択するだけでセット完了
- 「合成する」ボタンで **A3横サイズ** に自動整形して合成
- 生成された画像はそのまま **ローカルに保存可能**
- 画像データはサーバーに保存されない安全設計
- インストール不要（ブラウザのみで動作）

## 🖼 使用方法

1. 画面上の2つの画像枠に、合成したい画像をアップロード
2. そのまま「合成する」ボタンをクリック
3. A3横サイズに整えた合成画像が生成されます
4. 合成後は自動的に画像がダウンロードされます。
（ブラウザの設定により、ダウンロードフォルダが自動で開く場合があります）
